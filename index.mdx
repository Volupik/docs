# Welcome to FeedPin

FeedPin lets your clients pin visual feedback directly on your website. AI coding agents (Claude Code, Cursor, Windsurf) read and fix it automatically via MCP.

## How It Works

1. **Install the widget** — Add one script tag to your website
2. **Clients pin feedback** — They click anywhere and leave comments (no account needed)
3. **AI reads via MCP** — Claude Code, Cursor, or Windsurf reads feedback directly
4. **You ship the fix** — Review the AI-generated fix and deploy
